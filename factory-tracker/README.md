# 流水线管理系统

工厂流水线进度追踪工具，工人用手机记录工序，管理员实时查看进度。

## 文件结构

```
factory-tracker/
├── index.html          # 入口页面（所有页面的 HTML 结构）
├── css/
│   └── style.css       # 所有样式（主题色、布局、组件）
├── js/
│   ├── data.js         # 数据层（批次数据、工序操作、工具函数）
│   └── app.js          # 逻辑层（页面切换、渲染、用户交互）
└── README.md           # 本文件
```

## 如何使用

直接用浏览器打开 `index.html` 即可运行。

- **工人端**：选"工人" → 输入姓名 → 选批次 → 操作工序
- **管理端**：选"管理员" → 密码 `1234` → 查看进度

## 常见修改需求

### 修改工序名称
打开 `js/data.js`，找到第一行：
```js
const STEPS = ['裁切', '印刷', '装订', '质检'];
```
改成你需要的工序即可，其他代码不用动。

### 修改管理员密码
打开 `js/data.js`，找到：
```js
const ADMIN_PASSWORD = '1234';
```

### 修改主题颜色
打开 `css/style.css`，找到 `:root { }` 里的变量，例如：
```css
--color-accent: #2563eb;  /* 改这里换主色调 */
```

### 对接真实后端（数据库）
目前数据存在浏览器内存里，刷新会丢失。
要持久化存储，需要把 `js/data.js` 里 `DB` 对象的方法
替换成真实的 API 请求（fetch），后端可以用 Node.js 或 Python。

## 技术说明

- 纯 HTML + CSS + JavaScript，无任何依赖库
- 可直接在微信内置浏览器打开使用
- 迁移到微信小程序只需重写 HTML/JS 部分，逻辑不变
